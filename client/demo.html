📊 Performance Profiling Instructions

  1. Open Chrome DevTools (F12 or Cmd+Option+I)
  2. Go to Performance tab
  3. Click "Record" button (or press Ctrl+E)
  4. Run a test using the buttons above
  5. Stop recording after the test completes
  6. Analyze the performance profile:
    • Main thread should remain mostly idle (green/grey)
    • Worker thread should show cryptographic activity
    • FPS should stay close to 60fps (16.67ms per frame)

Expected Result: With Web Worker enabled, the UI remains responsive at 60fps while heavy signing operations happen off the main thread.
