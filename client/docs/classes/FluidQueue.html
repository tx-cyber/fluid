    Class FluidQueue

    FluidQueue manages offline queueing of fee-bump transactions. It persists queued XDRs to localStorage and automatically retries them when internet connectivity is restored.

    Example

    const queue = new FluidQueue(client, {
      onSuccess: (id, res) => console.log("Sent!", id, res),
      onQueueCleared: () => console.log("All transactions sent!"),
    });

    // Queue a transaction (works offline too)
    await queue.add(signedXdr, false);
